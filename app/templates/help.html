Frequently Asked Questions

What is metadata and why should I remove it?

Metadata is hidden information in files that can reveal personal details like your location, device information, or creation time. Removing it protects your privacy when sharing files online.

Is my data safe?

Yes! Files are processed locally on our server and automatically deleted after processing. We never store your files permanently or share them with third parties.

What file formats are supported?

We support images (JPG, PNG, TIFF, RAW), documents (PDF, DOCX, XLSX), videos (MP4, MOV, AVI), and audio files (MP3, WAV, FLAC).

What's the difference between cleaning presets?

Full clean: Removes all metadata including GPS, author, comments, and technical data.
Basic clean: Removes sensitive data like GPS and author but keeps technical metadata.
Custom: Choose exactly what to remove.

Why can't I upload certain files?

We only accept supported file formats for security reasons. Check that your file extension matches our supported formats and is under the size limit.

Can I process multiple files at once?

Yes! You can upload up to {{ max_files }} files at once. They'll all be processed with the same cleaning preset.

Troubleshooting

Files won't upload

  • Check file size (max {{ max_file_size_mb }} MB per file)
  • Ensure file format is supported
  • Try refreshing the page

Cleaning takes too long

  • Large files take longer to process
  • Multiple files are processed sequentially
  • Try processing fewer files at once

Download doesn't start

  • Check your browser's download settings
  • Disable popup blockers
  • Try a different browser
